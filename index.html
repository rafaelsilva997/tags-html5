HTML5

Formatação de Texto

Negrito: <strong>
Itálico: <em>
Sublinhado: <u>
Tachado (riscado): <del>
Sobrescrito: <sup>
Subscrito: <sub>
Código-fonte: <code>
Destacado: <mark>
Citação curta: <q>]
Citação em bloco: <blockquote>
Texto pequeno: <small>
Texto pré-formatado: <pre>

Listas

Lista não ordenada: <ul><li>Item</li></ul>
Lista ordenada: <ol> <li>Item</li></ol>
Lista de definição: <dl><dt>Título</dt> <dd>Descrição</dd></dl>

Elementos de Tabela

<table> → Define uma tabela
<tr> → Define uma linha
<td> → Define uma célula
<th> → Define um cabeçalho de célula
<caption> → Define um título para a tabela
<thead>, <tbody>, <tfoot> → Estrutura da tabela

Elementos de Mídia

Imagem: <img src="imagem.jpg" alt="Descrição">
Áudio: <audio controls> <source src="audio.mp3" type="audio/mpeg"> </audio>
Vídeo: <video controls> <source src="video.mp4" type="video/mp4"> </video>
Iframe: <iframe src="https://exemplo.com"<>/iframe>

Elementos Interativos

Botão: <button>Click aqui</button>
Links: <a href="https://site.com">Clique aqui</a>
Formulários: <form> <input type="text"><input type="submit"> </form>
Detalhes ocultáveis: <details> <summary>Ver mais</summary> Informação oculta</details>

Elementos Estruturais

Cabeçalhos: <h1> até <h6>
Parágrafo: <p>
Divisão de conteúdo: <div>
Seção: <section>
Cabeçalho de página: <header>
Rodapé: <footer>
Navegação: <nav>
Artigo: <article>
Barra lateral: <aside>
