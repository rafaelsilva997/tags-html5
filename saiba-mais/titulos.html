Para que serve ?

A tag <h1> é usada em HTML para definir o título principal de uma página ou seção. Ela representa o nível mais alto de cabeçalho e é geralmente usada para indicar a informação mais importante. Além disso, os mecanismos de busca como o Google dão mais peso ao conteúdo dentro de uma <h1>, tornando-a essencial para SEO (Otimização para Motores de Busca).

Outros Níveis de Cabeçalho:

O HTML possui seis níveis de cabeçalho, do <h1> ao <h6>, onde <h1> é o mais importante e <h6> o menos relevante:

<h1> - Título Principal

<h2> - Subtítulo

<h3> - Seção menor

<h4> - Subseção

<h5> - Detalhes
<h6> - Informação menos relevante
