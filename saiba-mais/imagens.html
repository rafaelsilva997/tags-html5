Como Adicionar uma Imagem ?

Para adicionar uma imagem em um documento HTML, use a tag <img.> Essa tag não precisa de fechamento e requer pelo menos o atributo src, que define o caminho da imagem.

Explicação:

src: Define o caminho da imagem (pode ser um arquivo local ou um link da web).

alt: Texto alternativo que aparece caso a imagem não carregue (importante para acessibilidade e SEO).
