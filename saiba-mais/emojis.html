Como adicionar emojis ?

Você pode adicionar emojis em HTML de duas maneiras principais:

1º - Usando o próprio caractere do emoji

2º - Usando códigos Unicode

O emoji 😊 tem o código U+1F60A, então no HTML ele fica &#x 1F60A;

O emoji 🌍 tem o código U+1F30D, então no HTML ele fica &#x 1F30D;
